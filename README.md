# playvibes

Une plateforme (web/app) où :

- Les utilisateurs partagent leurs playlists Spotify avec un titre, une description, un style, une humeur ou un contexte (ex. : "Focus Deep Work", "Afro Chill Night", "Rainy Day Coding").

- Les visiteurs peuvent découvrir de nouvelles playlists, filtrer par genre, ambiance, activité, pays, ou influenceur.

- Les utilisateurs peuvent écouter directement via l'API Spotify sans quitter le site.
